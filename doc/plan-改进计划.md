# 改进计划 · Ziang 学习练习站

> 基于「二年级小朋友视角 Review」制定
> 日期：2026/06/22
> 原则：让小朋友**真的看得懂、真的在练、真的想回来**

---

## P0 — 紧急（直接影响学习效果，不改 = 白练）

### 1. 阅读模块：加拼音标注

| 项目 | 内容 |
|------|------|
| **问题** | 短文含大量超纲字（耕、矛、盾、愚等），二年级识字量约 1600 字，遇到不认识的字只能跳过猜大意，导致"读了但没懂" |
| **方案** | ① 给生字加 `<ruby>` 拼音标注；② 或提供"点字读音"功能；③ 退一步：至少接入 TTS 朗读（复用口语模块的 `speak()` 接口） |
| **效果** | 小朋友不再因为不认字而放弃阅读，阅读模块从"摆设"变成"真练习" |
| **工作量** | 中等（2-3天）—— 需要对题库中的短文标注生字拼音，或开发点击读音组件 |

### 2. 英语口语：重写干扰项

| 项目 | 内容 |
|------|------|
| **问题** | 错误选项是"I am lesson""Lesson is good"这种语法乱凑的废话，小朋友一眼识破，4选1变1选1，完全没在练辨析 |
| **方案** | 重写题库中所有错误选项，改为**语法正确但语义不匹配**的真实英语句子，如用"Which page is it?"干扰"Which lesson is it?" |
| **效果** | 小朋友需要真正理解句意才能选对，从"猜笑话"变成"练听力+理解" |
| **工作量** | 中等（2-3天）—— 约 200 道题需要重写 3 个错误选项，可用脚本辅助生成后人工校对 |

### 3. 口算答错时显示正确答案

| 项目 | 内容 |
|------|------|
| **问题** | 答错后弹窗只说"没关系～🌈"就跳下一题，没显示正确答案。等到结果页时已经忘了当时在算什么 |
| **方案** | 错题反馈弹窗中增加"正确答案是 XX（你写的：YY）"，停留 1.5~2 秒再跳转 |
| **效果** | 即时纠正 > 延迟纠正，小朋友能当场记住易错点 |
| **工作量** | 小（半天）—— 修改反馈弹窗模板，增加答案显示和延时 |

### 4. 满分进度提示

| 项目 | 内容 |
|------|------|
| **问题** | 首页和科目页都没显示"今天已满分几次、还差几次抽转盘"，进度感缺失，动力不足 |
| **方案** | 在结果页 & 科目选择页显示 `🌟 今天满分 2/3 次，再 1 次就能转盘了！` |
| **效果** | 可视化进度 → 清晰目标 → 持续动力 |
| **工作量** | 小（半天）—— 读取 localStorage 中当日满分计数，渲染进度条/文字 |

## P1 — 重要（提升趣味性和适龄性，改了会明显更好）

### 5. 口算难度分级优化

| 项目 | 内容 |
|------|------|
| **问题** | ① "20以内"会出1+1这种幼儿园题；② 20直接跳到100，缺少50以内过渡；③ 没有"进位/退位"专项 |
| **方案** | ① a、b 最小值设为 3 或 保证和 ≥ 10；② 增加"50以内"档位（20→50→100）；③ 新增"进位加/退位减"专项模式 |
| **效果** | 题目难度贴合二年级教材进度，练到真正薄弱的地方 |
| **工作量** | 中等（1-2天）—— 修改 `generateQuestions()` 逻辑，增加模式选项 UI |

### 6. 阅读题目优化：问事实不问道理

| 项目 | 内容 |
|------|------|
| **问题** | 成语故事题目考"这个故事告诉我们什么道理"，抽象概念超出二年级认知水平 |
| **方案** | ① 优先出事实题（谁做了什么、为什么、结果怎样）；② 保留道理题时把答案改写为具体描述（"不能想着不做事就得到东西" 替代 "不能妄想不劳而获"） |
| **效果** | 降低理解门槛，让小朋友"读懂就能答对"而不是"猜抽象概念" |
| **工作量** | 中等（1-2天）—— 修改题库 JSON，重写部分题目和选项 |

### 7. 英语情景化改造

| 项目 | 内容 |
|------|------|
| **问题** | 中文 prompt 像翻译题（"想告诉妈妈你在学校："），缺乏情境感 |
| **方案** | 改用对话框/场景描述格式，如：`妈妈打电话问：「Ziang, where are you?」你该怎么回答？` |
| **效果** | 从"翻译选择"变成"情景应答"，更贴近真实语言使用场景 |
| **工作量** | 中等（1-2天）—— 重写约 200 道题的 prompt 描述 |

### 8. 错题列表优化

| 项目 | 内容 |
|------|------|
| **问题** | 结果页错题列表全部展开占满屏幕，二年级专注力有限看不完 |
| **方案** | 默认只展开错题，正确的折叠为"✓ 7 题全对"一行；错题重练按钮更突出 |
| **效果** | 聚焦错题，减少信息过载，家长陪复盘也更高效 |
| **工作量** | 小（半天）—— CSS + JS 折叠逻辑 |

### 9. 英语题目难度递进

| 项目 | 内容 |
|------|------|
| **问题** | 题目 shuffle 随机抽取，可能第一题就抽到较难句型，没有爬坡过程 |
| **方案** | 按 tag 设定难度等级（问候 < 感谢 < 道别 < 请求 < 学校 < 情绪），首次从简单开始，后续根据正确率智能调整 |
| **效果** | 渐进难度 → 建立信心 → 持续挑战，不会一上来就劝退 |
| **工作量** | 中等（1天）—— 题库增加难度字段，修改抽题算法 |

## P2 — 优化（锦上添花，让体验从"好"到"很好"）

### 10. 转盘奖励多样化

| 项目 | 内容 |
|------|------|
| **问题** | 转盘 6 格全是游戏时间（5-20分钟），差距不大，每次抽中感觉差不多 |
| **方案** | 奖励多样化：🎮 游戏时间 / 🍦 选零食 / 🎬 看动画 / 📚 选绘本 / 🤝 桌游时间 / ⭐ 称号"今日小天才" |
| **效果** | 每次抽奖充满期待，奖励覆盖不同偏好，家庭互动增加 |
| **工作量** | 小（半天）—— 修改转盘配置数组，可设为家长可配置 |

### 11. 交卷误触防护

| 项目 | 内容 |
|------|------|
| **问题** | "交卷📝"和"⌫删除"按钮位置相近，小朋友手指大容易按错，直接跳结果页 |
| **方案** | 交卷前加二次确认弹窗；或将最后一题的交卷按钮做得更显眼且远离删除键 |
| **效果** | 避免误操作导致的挫败感 |
| **工作量** | 小（2小时）—— 增加确认对话框 |

### 12. 成长进度系统

| 项目 | 内容 |
|------|------|
| **问题** | 星星只有 1-5 颗，没有长期成长感，得 2 颗就觉得"自己很差" |
| **方案** | 保留单次星星评价，额外增加"累计星星 → 升级"的成长进度条（如每 20 颗星升一级，解锁新头衔/主题） |
| **效果** | 长期激励，每天练习都有"涨经验"的感觉 |
| **工作量** | 中等（1-2天）—— 设计等级体系，增加 localStorage 存储，渲染进度条 |

### 13. 转盘视觉强化

| 项目 | 内容 |
|------|------|
| **问题** | 指针是黑色 ▼，不够显眼；抽中后对应格子没有高亮反馈 |
| **方案** | 指针变红 + 加发光效果；结果时对应格子放大 + 闪烁动画 |
| **效果** | 抽奖仪式感拉满，小朋友更兴奋 |
| **工作量** | 小（半天）—— CSS 动画 + JS 高亮逻辑 |

### 14. TTS 发音优化

| 项目 | 内容 |
|------|------|
| **问题** | `SpeechSynthesisUtterance` 在不同设备发音质量差异大，可能误导发音学习 |
| **方案** | ① 加载时检测英文 voice，优先选高质量（macOS Samantha、iOS 增强语音）；② 提供家长设置语音的入口；③ 远期：预录音频 |
| **效果** | 发音标准统一，小朋友不会学到"机器人腔" |
| **工作量** | 小-中等（半天~1天）—— voice 检测 + 设置页 |

### 15. "先读再答"引导

| 项目 | 内容 |
|------|------|
| **问题** | 阅读模块可以不看文章直接看选项蒙答案 |
| **方案** | 进入题目时选项隐藏 5 秒（显示"先读文章哦~⏳"），或加"我读完了 ✅"按钮再显示选项 |
| **效果** | 引导认真阅读，而不是猜答案 |
| **工作量** | 小（2-3小时）—— 定时器 + 显示/隐藏逻辑 |

## 衡量标准

| 指标 | 当前估计 | 目标 |
|------|----------|------|
| 每日主动打开次数 | 1-2 次（需家长催） | 3+ 次（自己想练） |
| 单次练习时长 | 5 分钟 | 10-15 分钟 |
| 阅读模块使用率 | 低（看不懂就跳过） | 与口算持平 |
| 英语答对率 | 95%+（太简单） | 70-85%（合理挑战） |
| 转盘触发率 | 偶尔 | 每天至少 1 次 |
