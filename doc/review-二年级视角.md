打开网站，深紫色加粉色的夜空背景配上一闪一闪的小星星，再加上中间晃来晃去的 ✨ 表情，**第一秒就把我吸住了**。这种感觉就像打开一本漫画书，不像在做作业。

- 三张大卡片：⚡口算、📖阅读、🗣️日常口语，**emoji 一眼就知道是干嘛的**，不用爸爸教。
- "第3次满分可抽奖 🎡"——立刻好奇：转盘是什么？我要赶紧得满分！

**亮点小结：** 颜值、动效、命名感都很顶，二年级小朋友会愿意主动打开。

## 二、点进口算之后 ⚡

### 😊 喜欢的地方

- 大键盘按下去会"陷下去"再弹回来（`box-shadow: 0 2px 0 #ddd` + `:active`），**像在按真按键**，手感超棒。
- 答对了有 🎉、答错了有 🌈，**而且都说"没关系～""下一题！"**，不会骂我，我不会害怕做错。
- 连对两题以上会弹"🔥 连对 X 题！"——**像打游戏一样上分**，我会想再多对几题。
- 计时小框框是浅蓝色的，不是红色那种紧张的颜色，**不会让我心慌**。
- 切到别的 tab、看到反馈弹窗时计时会暂停，**不会"被偷时间"**（虽然我也不知道是怎么做到的，反正爸爸说这个很贴心）。

### 😣 困惑或不喜欢的地方

1. **"二十以内"还能接受，但题目是 1+1、2+3 这种我闭着眼都会的**
   `generateQuestions()` 里 `rand(1, cap-1)` 让 a 从 1 开始，所以会出现 "1 + 2" 这种**幼儿园题**。二年级了，10 以内已经太简单，但题库会随机给我出。
   - 👀 建议：起码 a、b 都 ≥ 2，或者**和 ≥ 10**（除了第一次教学），不然得 10 分但完全没成就感。

2. **"100 以内"难度跳跃太大**
   从 20 一下到 100，但 20-99 之间不存在选项。**二年级上学期一般在 50 以内**，下学期才到 100。
   - 👀 建议：增加"五十以内"选项，或者按教材进度做"20→50→100"三档。

3. **"加减混合"是一个选项，但没有"专练进位/退位"**
   二年级老师反复讲的就是 **进位加（8+7）** 和 **退位减（13-7）**。现在的随机模式会大量出"凑十法"不需要的题（10+5、20-3）。
   - 👀 建议：加一个"进位/退位"专项模式，让我**练我真正不会的**。

4. **数字键盘只有 0–9，没有切换正负号**
   万一题目算出负数怎么办？现在题库保证不会出（`b = rand(1, a)`），所以**目前没问题**，但万一以后扩展就要注意。

5. **"下一题 ➡️" 和 "交卷 📝" 的按钮位置和"⌫ 删除"挨得很近**
   小朋友手指比较大，按错的可能性不低，尤其想删数字时按到"交卷"会直接跳到结果页。
   - 👀 建议：交卷前加一个二次确认，或者**最后一题的"交卷"做得更显眼+离删除键远一点**。

6. **"心算后填答案"这一行小灰字**，小朋友会下意识忽略
   提示用 `#bbb` 太淡了，建议用稍微醒目一点的颜色或者改成 emoji 提示。

## 三、点进阅读之后 📖

### 😊 喜欢的地方

- 黄底+左边一道金色边的"羊皮纸"风格短文卡片，**像故事书里的小盒子**，看着就想读。
- 顶部"成语故事 / 生活常识"小标签，**让我一眼知道这篇是故事还是知识**。
- 不计时！**这是最棒的设计**——读得慢不会被催，我可以认真读。
- 题目里能优先抽没读过的（`pickQuestionsSmart`），**不会一直重复**。

### 😣 困惑或不喜欢的地方

1. **文章字太多，没有拼音**——这是**最严重**的问题
   - 100~129 字的文言风短文，里面有"宋国""农夫""毫不费力""耕作""庄稼""疑神疑鬼""惯偷""自相矛盾"……
   - 二年级**识字量大约 1600 字左右**，且很多是看图识字。遇到"耕""矛""盾""愚""矛"这些字我**真的不认识**，只能跳过去猜大意。
   - 👀 建议（P0）：
     - 给生字加**拼音 ruby 标注**（`<ruby>耕<rt>gēng</rt></ruby>`），或者
     - 提供"**点字读音**"功能（点哪个字读哪个）
     - 退一步：至少让短文支持 **TTS 朗读**（按一下听一遍），跟口语模块用同一个 `speak()` 接口

2. **成语故事的"道理"对二年级太抽象**
   像「掩耳盗铃 → 自己欺骗自己」、「亡羊补牢 → 出了问题及时补救还来得及」，这些**抽象概念**二年级理解起来很费力。题目还把"道理"作为正确答案，等于考的是**抽象思维**而不是阅读。
   - 👀 建议：
     - 题目尽量**优先问事实**（谁、什么、为什么），少问"告诉我们什么道理"
     - 或者把抽象答案改写得更具体：「不能想着不做事就得到东西」比「不能妄想不劳而获」对小朋友友好得多

3. **每篇都有"正确答案在第一项 0"被脚本轮换了，但 4 个选项长度差太大**
   像「自相矛盾」的选项：
   - "他的话前后矛盾"（6 字）
   - "他不会算账"（5 字）
   - "没有人买他的东西"（8 字）
   - "他忘记了"（4 字）
   长度差距让聪明的小朋友会用"猜最长的"作弊。
   - 👀 建议：让 4 个选项长度尽量接近，**避免长度暴露答案**。

4. **没有"读完才能选"的逻辑**
   现在可以直接看到题目和选项就猜，不读文章也能蒙对。
   - 👀 建议（轻量）：增加"先读再答"的引导，比如**进入题目时先把选项隐藏 5 秒**，强制看文章；或者加一个"我读完了 ✅"按钮再亮出选项。

5. **生活常识题里有些选项太"陷阱"**
   比如刷牙题，正确答案是"……（标准回答）"，错误项里有"早晚各刷一次"这种**看起来也对**的。二年级的判断力没那么细，容易引发挫败。
   - 👀 建议：错误选项尽量"**明显不对**"，让小朋友通过"排除明显错"就能得分，建立信心。

## 四、点进日常口语之后 🗣️

### 😊 喜欢的地方

- **进入题目自动朗读**（350ms 后 `speakEnglish(q.heard)`）——耳朵直接听到，超棒！
- 每个选项后面有 🔊 按钮可以**单独听任意一个选项**——这功能我爱死了，能反复练发音。
- 答对了会朗读"正确答案"，**正反馈直接绑定在英语句子上**。
- 题目分成"问候/感谢/介绍/学校/饮食/情绪"等 10 个标签，**贴近小朋友生活**。

### 😣 困惑或不喜欢的地方

1. **错误选项里的"陷阱"是死板的语法错** —— 这是**最大的体验问题**
   抽样看到的几乎每道题都长这样：
   ```
   想问问今天学第几课：
   ✗ I am lesson.
   ✓ Which lesson is it?
   ✗ Lesson is good.
   ✗ Good morning!
   ```
   小朋友一看就知道 "I am lesson" 是搞笑的（我是课？哈哈），等于**4 选 1 变成 1 选 1**。这就破坏了练习意义：
   - 第一次会觉得"这英语好简单"，建立**虚假自信**
   - 几道之后就**腻**了，因为根本没在思考
   - 👀 建议（P0）：错误选项要写得**像真的英语**，比如：
     ```
     ✗ Which page is it?
     ✓ Which lesson is it?
     ✗ Where is the lesson?
     ✗ How is the lesson?
     ```
     这才是**真在练辨析**。

2. **prompt 写得有点像翻译题**
   "想告诉妈妈你在学校："这种中文 prompt 让题目变成"翻译 + 选项匹配"，而不是真正的语境理解。**情境感**不够。
   - 👀 建议：用更"情景化"的描述，比如配图、或者用对话框格式：
     ```
     妈妈打电话过来：「Ziang, where are you?」
     你该怎么回答？
     ```

3. **TTS 发音依赖系统**
   `SpeechSynthesisUtterance` 在不同手机/浏览器里发音差异极大。家长机器上发音可能听起来很怪，**影响小朋友学发音**。
   - 👀 建议：
     - 在加载时检测可用的英文 voice，**优先选高质量的**（如 macOS 的 "Samantha"、iOS 的增强语音）
     - 提供"选择发音"的小开关让家长设置一次
     - 或考虑**预录音频**（200 句不算多，10MB 内能装下）

4. **题目顺序固定难度递增不足**
   现在是 shuffle 随机抽取，但小朋友可能第一题就抽到 `e150` 这种较难的句子，**没爬坡过程**。
   - 👀 建议：按 tag 难度分级（问候 < 感谢 < 道别 < 请求 < 学校 < 情绪），第一次玩从最简单开始。

## 五、答对之后的反馈 🎉

### 😊 喜欢的地方

- 答对了 🎉 + 浅绿色边框闪一下，**画面感很足**。
- 反馈弹窗 0.45 秒就消失，**不打断节奏**。
- 80% 以上正确率会下五彩纸屑！

### 😣 困惑或不喜欢的地方

1. **反馈太短，错的时候没看清正确答案**
   口算答错时，弹窗只显示"没关系～"加一个 🌈，**没有显示正确答案**。然后就直接进下一题。我只能在最后结果页才能看到自己错在哪——但那时**我已经忘了我当时算的是什么**。
   - 👀 建议（P0）：错题反馈时**显示正确答案**至少 1.5 秒，比如：
     ```
     🌈
     正确答案是 15
     （你写的：14）
     ```

2. **没有"重做这题"的选项**
   想再做一次同样的题不行，只能再练一组。错题重练要等到这组结束。

## 六、结果页 📝

### 😊 喜欢的地方

- 分数会从 0 跳到目标值的**动画**——超有仪式感！
- ⭐ 星星数随得分变化，**5 颗星是大目标**。
- "🎉 新纪录！"会跳出来——**有竞争感**。
- 错题列表会列出每道题的"你答的 / 正确答案"，**家长可以陪我看**。

### 😣 困惑或不喜欢的地方

1. **星星只有 1~5 颗，没有等级感**
   1 颗星到 5 颗星跨度太大，**得了 2 颗就觉得自己很差**，不会想再来一次。
   - 👀 建议：保留星星，**额外加一个"成长进度条"**（累计星星 → 升级），让我看到自己越来越强。

2. **错题列表太长，二年级看不完**
   10 道题展开占满屏幕，二年级专注力不够，**家长看着也累**。
   - 👀 建议：**默认只展开错题**，对的折叠成"✓ 7 题全对"一行。

3. **"再练一组 🔄"和"错题重练 💪"按钮颜色相近**
   都是高饱和粉色/橙色，**看不出主次**。错题重练应该更突出，因为它是"针对性练习"。

## 七、转盘奖励 🎡

> 这是整个项目我最想夸的设计。

### 😊 喜欢的地方

- "**今天第 3 次满分**触发"——给了我一个**清晰的目标**，不是无脑刷分。
- 奖励是"**5~20 分钟游戏时间**"，比"100 金币 = 0.5 元"那种**实在**得多。
- 需要"**家长确认**"按钮，避免我自己悄悄抽完跑去玩游戏。
- 转一次就锁定到第二天，**不会让我整天疯狂刷**。

### 😣 困惑或不喜欢的地方

1. **转盘只有 6 格而且都是游戏时间**
   时间长短没有大跨度，5~20 分钟差距不大，**每次抽中都差不多**。
   - 👀 建议：奖励多样化，比如：
     - 🎮 游戏时间 5/10/15/20 分钟
     - 🍦 选今天的零食
     - 🎬 看一集动画片
     - 📚 选一本绘本/睡前故事
     - 🤝 全家一起玩一个桌游
     - ⭐ 称号"今日满分小天才"

2. **转盘的指针不太显眼**
   黑色的 ▼ 在白色边框上，第一次转完不知道指向了哪格——以为是中间的"5 分钟"，结果是另一个。**视觉反馈不够强**。
   - 👀 建议：指针变红 + 加发光，结果时**对应格子放大+闪烁**。

3. **"为什么是第 3 次"**
   小朋友不理解为什么不是"每次满分都抽"或"第 1 次满分就抽"。
   - 👀 建议：在家用模式可以让家长配置（1/3/5 次），默认仍是 3。

4. **没有"距离下一次抽奖还剩 N 次满分"的提示**
   首页或科目页都没显示"今天已满分 2 次，再 1 次就能转盘了！"——**没有进度感**。
   - 👀 建议（P0）：在结果页 & 科目页**显示当天满分进度** `🌟 今天满分 2/3 次`。

## 八、整体使用感受总结

### 我打分 ⭐⭐⭐⭐（4/5 颗星）

| 维度 | 评分 | 一句话 |
|------|------|--------|
| 视觉好看程度 | ⭐⭐⭐⭐⭐ | 像漫画书，紫色星空超酷 |
| 上手容易程度 | ⭐⭐⭐⭐⭐ | 三个 emoji 卡片，一秒看懂 |
| 题目难度合适 | ⭐⭐ | 口算太简单，阅读字太多，英语干扰项太蠢 |
| 趣味性 | ⭐⭐⭐⭐ | 转盘 + 连击 + 满分纸屑都很赞 |
| 学习效果 | ⭐⭐⭐ | 阅读看不懂、英语没在练，效果打折 |
| 想不想再来一次 | ⭐⭐⭐⭐ | 转盘那天会很想刷，平时还行 |

### 最大的三个痛点

1. **🟥 阅读没拼音，识字量不够的小朋友读不懂** — 直接劝退
2. **🟥 英语干扰项太蠢，4 选 1 变成 1 选 1** — 失去练习意义
3. **🟧 口算题目偏简单，没有"进位/退位"专项** — 对得起 20 分，对不起教材

### 最大的三个亮点

1. **🟩 转盘奖励 + 家长确认 + 满分进度机制** — 这是真正打通"动机循环"的设计
2. **🟩 英语自动朗读 + 单选项可点 🔊** — 学发音的体验已经超过很多 App
3. **🟩 整体视觉、动效、emoji 用法** — 完全击中小朋友审美

### 一句话给开发者

> "这个网站已经做得很好啦！只要把**'让我真的看得懂、真的在练'**这件事再补一补，就是我每天都想打开的练习站。"

——来自 8 岁的 Ziang
